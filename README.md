## Acerca del Proyecto

Esta es una aplicación Laravel diseñada para la gestión de visitantes. Construida con Laravel Framework v12.33.0.

## Instalación

1. Clona el repositorio
2. Ejecuta `composer install`
3. Ejecuta `npm install`
4. Copia `.env.example` a `.env`
5. Ejecuta `php artisan key:generate`
6. Ejecuta `php artisan migrate`
7. Ejecuta `npm run build`

## Uso

Para iniciar el servidor de desarrollo:
```bash
php artisan serve
```

Para desarrollo con hot reload:
```bash
npm run dev
```
